# ICIR 加权方法论

## 加权流程

### Step 1: 因子预处理
对每个因子做行业中性 + 市值中性 + 标准化（z-score）。

### Step 2: 计算 ICIR
滚动12个月计算 IC（因子值与未来N日收益的秩相关系数），近3月衰减加权。

```
IC_t = rank_corr(factor_t, return_{t+N})
ICIR = mean(IC_rolling) / std(IC_rolling)
```

### Step 3: 大类内权重归一化
大类内按 ICIR 归一化：`w_i = ICIR_i / sum(ICIR_i)`

### Step 4: 风险平价（大类间）
```
权重_i ∝ 1 / Var(IC_i)
```
或约束：每个因子贡献相等边际风险贡献（ERC）。

### Step 5: 固定比例 + 风险平价结合
大类按固定比例（用户选择策略等权） × 风险平价修正。

### Step 6: 非线性交互项
加分项：
- 低波 × 高动量（低波动环境下动量更有效）
- 质量 × 成长（高质量 + 高成长 = 双击潜力）
- 低估值 × 情绪（低估值 + 情绪回暖 = 戴维斯双击）

### Step 7: 综合得分
```
综合得分 = Σ(权重_i × 因子_i_zscore) + 交互项加分
```

### Step 8: 筛选与调仓
- 选前 Top 50-80 只
- ICIR < 0.3 的因子自动剔除（不参与打分）
- 月度调仓

## 最优化目标

```
max IR = (wᵀμ) / sqrt(wᵀΣw)
s.t.
  sum(w) = 1
  w ≥ 0（非负）
  行业/市值/Beta 暴露 < 阈值（中性约束）
```
